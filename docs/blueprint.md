# **App Name**: HealthWise AI

## Core Features:

- Multilingual Support: Enable the chatbot to respond in multiple languages, catering to diverse user demographics.
- Symptom Checker: AI-driven tool that evaluates user-reported symptoms to provide potential health concerns.
- Preventive Care Info: Disseminate actionable info regarding disease prevention measures.
- Vaccination Schedules: Provide up-to-date vaccination schedules and reminders according to age and location.
- Real-Time Health Alerts: Provide immediate notifications on disease outbreaks in the user's area.
- WhatsApp/SMS Integration: Make the chatbot accessible via WhatsApp and SMS for broader reach.

## Style Guidelines:

- Primary color: Fresh green (#6AB475), evoking health and nature.
- Background color: Light green (#E6F2E8), for a calm and clean feel.
- Accent color: Soft yellow (#D4AC0D), for CTAs, draws attention without being alarming.
- Body and headline font: 'PT Sans' (sans-serif), combining modernity and approachability for comfortable reading on all devices.
- Use simple, recognizable icons for health topics and actions.
- Simple, clear layout optimized for mobile devices.
